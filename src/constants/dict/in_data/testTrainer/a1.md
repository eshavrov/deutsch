
## GRAMMATIK
### Verb: Konjugation 

|           | kommen | heißen | sein |
| --------- |:-------|:-------|:-----|
| ich       | komme  | heiße  | bin  |
| du        | kommst | heißt  | bist |
| Sie       | kommen | heißen | sind |
| er/sie/es | kommt  | heißtt | ist  |

---
### W-Frage: wer, wie, woher 

|       | Position 2 |      |
| ----- |:-----------|:-----|
| Wer   | ist        | das? |
| Wie   | heißen     | Sie? |
| Woher | kommst     | du?  |
